# Gemini-authored plans (org pack)

Plans **`cco`** writes for **repo-local** work belong in **`<project>/.gemini/docs/plans/`** (example: `content-foundry` vault).

Globally stowed org plans land under **`~/.gemini/docs/plans/`** after DotMate stow.

Use date-stamped filenames: `YYYY-MM-DD-slug.md`.
